# Portfolio Requirements

**Computer Science and AI Pathway — Student Hub**

---

## Your Portfolio Has Two Layers

### 1. GitHub Repository (Required)

Your personal portfolio repo should include:

```text
student-portfolio/          (or your chosen name)
├── README.md               ← who you are, what you're learning
├── projects/               ← one folder per major project
├── reflections/            ← weekly and project reflections
├── ai-literacy/            ← AI literacy evidence
├── ai-math-bridge/         ← optional math bridge labs
├── design/                 ← Figma links and design notes
└── full-stack-practice/    ← capstone and integration work
```

### 2. Notion Portfolio (Recommended)

A public Notion page linking to GitHub:

- Home
- Projects
- Skills
- AI Literacy
- Reflections
- GitHub Links

See your teacher’s Notion setup guide if provided.

---

## What Each Project Folder Must Include

| Item | Required |
|---|---|
| `README.md` | What the project does, how to run it, what you learned |
| Source code | Your work, committed over time |
| Reflection | Challenges, AI use disclosure, next steps |
| Evidence | Screenshots, demo link, or test output if applicable |

---

## Quality Standards

- README is written **by you**, not copied from AI without understanding
- Commits show **progress over time**, not one giant upload
- You can **explain** your code orally when asked
- AI-assisted sections are **disclosed** in reflections

---

## Capstone Evidence

For the AI School Assistant (or equivalent final project):

- Working demo (local or deployed)
- Document ingestion + Q&A with sources
- Final presentation or showcase recording
- Complete reflection on responsible AI use
