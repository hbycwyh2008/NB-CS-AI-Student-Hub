# Week 1 Agenda: GitHub, Markdown, and Your First Web Page

**Computer Science and AI Pathway**

---

## Learning Goals

By the end of Week 1, you should be able to:

- explain what a GitHub repository is
- create or update a README file
- use Markdown headings, lists, links, and image syntax
- build a simple HTML and CSS profile page
- submit evidence using GitHub links, screenshots, and reflection

## Reflection Requirement

At the end of each assignment, complete the reflection file in your assignment repository.

A good reflection explains:

- what you built
- what was confusing
- what you fixed
- how you used AI, if you used it
- what you can now do without copying
