# Student Workflow Test Checklist

Use this checklist before teaching Week 1.

## Test 1: Open Student Hub

- [ ] Open the Student Hub README.
- [ ] Open Week 1 Agenda.
- [ ] Confirm the three template repository links are visible.

## Test 2: Open template repository

- [ ] Open Assignment 1 template repository.
- [ ] Confirm the repository is accessible from a student account or incognito browser.
- [ ] Confirm the **Use this template** button is visible.

## Test 3: Create student assignment repo

- [ ] Click **Use this template**.
- [ ] Create a new repository named `assignment-01-github-readme-test`.
- [ ] Confirm the starter files copied correctly.

## Test 4: Complete a minimal assignment

- [ ] Edit one required file.
- [ ] Commit the change with a meaningful commit message.
- [ ] Complete or update `reflection.md`.

## Test 5: Submit evidence

- [ ] Copy the student assignment repository link.
- [ ] Submit the link using the Week 1 submission form.
- [ ] Confirm the teacher can open the submitted repo link.

- [ ] Workflow works.
- [ ] Student cannot access template repo.
- [ ] Submission form missing.
- [ ] Reflection file unclear.
