# Course Overview

**Computer Science and AI Pathway — Student Hub**  
**Full-Stack Web and AI Application Development**

---

## What This Course Is

A **portfolio-first**, **evidence-based**, **mastery-focused** course in full-stack web and AI application development.

You will build real projects, document your process on GitHub, and show that you can **explain, rebuild, debug, modify, and apply** what you learn — not just watch, copy, or run code once.

**Final project:** AI School Assistant — a document-grounded Q&A web app with sources. Your teacher will share the student-facing specification when you reach the capstone.

---

## Learning Path (Overview)

```text
Git & GitHub → Portfolio → AI Literacy → Web Foundations
→ JavaScript → React → Next.js → FastAPI → Full-Stack Integration
→ Capstone → Final Showcase
```

---

## Course Philosophy

> This is not only an AI literacy course. It is an **AI application engineering** course built on top of AI literacy.

You will move from watching tutorials to **building, documenting, publishing, and explaining** real projects.

---

## What Success Looks Like

This course is not designed for students who only watch demonstrations, copy solutions, or use AI to complete tasks without understanding them.

You are expected to:

- Write and test your own code
- Fix errors step by step
- Explain your work in your own words
- Submit honest reflections and AI use disclosures

**Running code once is not enough.** See [Weekly_Agenda_Template.md](Weekly_Agenda_Template.md) for how a typical class works and what mastery means.
