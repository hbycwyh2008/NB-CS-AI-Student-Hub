# Teacher Template Workflow Checklist

**For teachers — setup instructions only (no private curriculum content)**

---

## Step 1: Confirm template repositories

Confirm these template repositories exist:

- [hbycwyh2008/lesson-01-github-readme-template](https://github.com/hbycwyh2008/lesson-01-github-readme-template)
- [hbycwyh2008/lesson-02-markdown-portfolio-template](https://github.com/hbycwyh2008/lesson-02-markdown-portfolio-template)
- [hbycwyh2008/lesson-03-html-css-profile-page-template](https://github.com/hbycwyh2008/lesson-03-html-css-profile-page-template)

---

## Step 2: Enable Template Repository Setting

For each template repository:

1. Open the repository.
2. Go to **Settings**.
3. Enable **Template repository**.

---

## Step 3: Add links to Student Hub

Template repository links are in [Week_01_Agenda.md](Week_01_Agenda.md).

Confirm students can open each link and see the **Use this template** button.

---

## Step 4: Prepare submission collection

Use one of these:

- Google Form
- Microsoft Form
- Notion database
- LMS assignment
- shared spreadsheet

Recommended fields:

- student name
- class
- assignment number
- GitHub repository link
- reflection link
- notes

Add the submission form link to the Student Hub when ready (for example in `Week_01_Agenda.md` or README).

---

## Step 5: Test as a student

Before class:

1. Open Student Hub.
2. Open [Week 1 Agenda](Week_01_Agenda.md).
3. Open Assignment 1 template repo.
4. Click **Use this template**.
5. Create a test student repository.
6. Edit a file.
7. Commit changes.
8. Submit the test repo link through your submission form.

---

## Boundary

Students should access:

- Student Hub
- assignment template repositories
- their own assignment repositories
- their own portfolio repositories

Students should **NOT** access:

- the private master curriculum repository
- teacher-only lesson plans
- answer keys
- private prompts
- internal planning notes

---

## Note on GitHub Classroom

This course does **not** use GitHub Classroom. Students use **Use this template** on GitHub to create their own assignment repositories and submit repository links manually.
